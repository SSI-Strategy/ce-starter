# Assignments

Ten starter assignments. Each directory is a standalone starting point — open its `index.html` in a browser for the interactive version, or read `README.md` on GitHub.

- [01 · CRO Time & Invoice Tracker](./01-cro-time-invoice-tracker/) — A local tool where a CRO project manager logs billable hours against studies, sees month-to-date spend by person, and compares it to a simple monthly budget.
- [02 · Competitive Vendor Bid Comparison](./02-vendor-bid-comparison/) — A tool to enter line-item bids from multiple vendors for the same scope of work and view them side-by-side with totals and per-line deltas.
- [03 · Protocol Deviation Log](./03-protocol-deviation-log/) — A log where monitors record protocol deviations observed at study sites, categorise them, and track each one from open to resolved.
- [04 · Study Site Feasibility Tracker](./04-site-feasibility-tracker/) — A tool for a feasibility lead to track candidate sites for a new study, record each site's survey answers, and rank them.
- [05 · Clinical Query Tracker](./05-clinical-query-tracker/) — A lightweight tracker for data-management queries raised against case-report-form data: who raised it, which subject and field, when it was answered, and how long it aged.
- [06 · Habit Tracker](./06-habit-tracker/) — A personal tool to define recurring habits, check them off daily, and see streaks and a weekly completion rate.
- [07 · Team Standup & Blocker Log](./07-standup-blocker-log/) — A tool a small team uses to post a short daily update and list their current blockers, and a screen that shows all open blockers across the team, sorted by age.
- [08 · Expense Splitter (lite)](./08-expense-splitter/) — A small group tracks shared expenses — who paid, how much, for whom — and the app computes who owes whom.
- [09 · Reading List](./09-reading-list/) — A personal tracker for books: want-to-read, reading, finished; with a rating and a short note per book, and a pages-per-week report.
- [10 · Agent Skill Library](./10-agent-skill-library/) — A catalogue of agent 'skills' (name, description, tags, when-to-trigger, example prompts) with search and filtering. A companion tool for the work this team is actually doing.

See [`ASSIGNMENTS.md`](../ASSIGNMENTS.md) at the repo root for the full catalogue and selection rationale.
